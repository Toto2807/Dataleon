# React + TypeScript + Vite

# Anglais

To launch the project, please type the command: “npm install” in your terminal.
Once the node_modules installation is complete, type: “npm run dev” to launch the project and click on the link that will appear in your terminal.

# Français

Pour lancer le projet, veuillez saisir la commande « npm install » dans votre terminal.
Une fois l'installation de node_modules terminée, saisissez « npm run dev » pour lancer le projet et cliquez sur le lien qui apparaîtra dans votre terminal.
